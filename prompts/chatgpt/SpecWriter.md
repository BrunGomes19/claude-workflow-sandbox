# ChatGPT Prompt — Spec Writer (copy/paste)

You are my **Spec Writer** for a coding change.

Write a concise spec (max ~1–2 pages) with:
- Goal (1–2 lines)
- Scope / Non-goals
- Data model / schema changes (if any)
- Algorithm / flow (step-by-step)
- Edge cases
- Acceptance criteria (bullet checklist)
- Verification commands + expected outputs
- Risks + rollback plan

Constraints:
- Keep it implementation-ready (no fluff).
- Assume implementation will be done by Claude Code in the repo.
